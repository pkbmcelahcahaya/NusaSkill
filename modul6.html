Modul 6: JavaScript & Logika Pemrograman

Tahap Ahli (Advance) | Fokus: Membuat Website Menjadi Hidup (Interaktif)

Instruktur: Admin Budi

Jika HTML adalah tulang, dan CSS adalah kulit, maka JavaScript adalah otot dan otaknya. JS membuat tombol bisa berfungsi, kalkulator bisa berhitung, dan web bisa merespon pengguna.

Variabel ibarat "kotak penyimpanan" untuk menyimpan informasi seperti nama, angka, atau status.

Mendeklarasikan Variabel (let & const)

let nama = "Budi"; // Teks (String)
let umur = 25; // Angka (Number)
const phi = 3.14; // Nilai tetap yang tidak bisa diubah

console.log(nama); // Menampilkan output ke Console Browser

ðŸ“… Minggu 2: Logika Percabangan (If / Else)

Sama seperti fungsi IF di Excel, JavaScript juga bisa mengambil keputusan berdasarkan kondisi.

let nilai = 85;

if (nilai >= 80) {
  alert("Selamat, Anda Lulus!");
} else {
  alert("Maaf, Anda Remedial.");
}

ðŸ“… Minggu 3: DOM (Document Object Model)

DOM adalah cara JavaScript berbicara dengan HTML. Kita bisa mengubah teks atau warna di web langsung dari kode JavaScript saat tombol diklik.

Mengambil Elemen HTML

// HTML: <h1 id="judul">Halo</h1>

let teks = document.getElementById("judul");
teks.innerHTML = "Teks ini diubah oleh JavaScript!";
teks.style.color = "red"; // Mengubah warna jadi merah
