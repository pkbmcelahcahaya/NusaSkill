ðŸ“… Minggu 1: Dasar Hardware & Software

Komputer ibarat tubuh manusia. Ada bagian fisik yang bisa disentuh (Hardware/Perangkat Keras) dan ada "pikiran" yang menjalankan perintah (Software/Perangkat Lunak).

1. Mengenal Komponen Fisik (Hardware)

  • Input (Pemasukan): Alat untuk memasukkan perintah. Contoh: Keyboard (untuk mengetik) dan Mouse (untuk menggerakkan kursor).
  • Proses (Otak Komputer): Berada di dalam kotak CPU/Laptop. Terdapat Prosesor yang berpikir, dan RAM (memori jangka pendek) yang membantu komputer berpikir cepat.
  • Storage (Penyimpanan): Hardisk atau SSD adalah memori jangka panjang. Tempat di mana foto, video, dan dokumen Anda disimpan.
  • Output (Keluaran): Alat untuk menampilkan hasil kerja. Contoh: Monitor (layar), Printer, dan Speaker.

2. Cara Memegang dan Menggunakan Mouse

Klik Kiri (Left Click): Tekan tombol kiri mouse 1x untuk memilih sesuatu. Tekan 2x dengan cepat (Double Click) untuk membuka aplikasi/file.

Klik Kanan (Right Click): Tekan tombol kanan mouse 1x untuk memunculkan Menu Pilihan Rahasia (seperti copy, paste, delete).

Scroll (Roda Tengah): Putar roda di tengah mouse untuk menggulir halaman ke atas atau ke bawah.

3. Menghidupkan & Mematikan Komputer yang Benar

Jangan pernah mematikan komputer dengan mencabut kabel listrik langsung! Ini bisa merusak sistem (Windows).

  • Shutdown: Mematikan komputer sepenuhnya. (Klik logo Windows di sudut kiri bawah > klik ikon Power âš¡ > pilih Shut down).
  • Sleep: Menidurkan komputer sejenak. Komputer mati tapi saat dinyalakan lagi, semua aplikasi yang belum ditutup akan kembali seperti semula.
  • Restart: Mematikan komputer lalu otomatis menyalakannya kembali. Sangat berguna jika komputer tiba-tiba hang/lag.

ðŸ“… Minggu 2: Desktop & Manajemen File Explorer

Setelah komputer menyala, layar pertama yang Anda lihat disebut Desktop. Desktop ibarat meja kerja digital Anda.

1. Bagian-Bagian Layar Utama

  • Ikon/Shortcut: Gambar kecil di layar untuk membuka aplikasi dengan cepat (Double Click).
  • Taskbar: Pita hitam panjang di paling bawah layar.
  • Start Menu: Tombol logo Windows di pojok kiri bawah. Berisi semua daftar aplikasi di komputer.

2. File Explorer (Lemari Penyimpanan Anda)

File Explorer (ikon berbentuk map/folder kuning di Taskbar) adalah tempat kita mencari, merapikan, dan menyimpan semua data.

Langkah Membuat Folder Baru:
1. Buka File Explorer.
2. Klik pada Documents (di menu sebelah kiri).
3. Di area yang kosong (warna putih), Klik Kanan mouse Anda.
4. Pilih New > lalu klik Folder.
5. Ketik nama foldernya (contoh: "Tugas NusaSkill"), lalu tekan tombol Enter di keyboard.

Cara Memindahkan Data:

  • Copy (Salin): Klik kanan file > pilih Copy > buka folder tujuan > klik kanan > Paste (Menempelkan). File lama tetap ada.
  • Cut (Pindah): Klik kanan file > pilih Cut > buka folder tujuan > klik kanan > Paste. File lama akan hilang dan pindah ke tempat baru.
  • Rename: Klik kanan file > pilih Rename untuk mengganti namanya.
  • Delete: Klik kanan file > pilih Delete untuk menghapus (masuk ke tempat sampah / Recycle Bin).

ðŸ“… Minggu 3: Pengaturan & Personalisasi (Settings)

Sekarang kita akan membuat komputer terasa lebih "Personal" milik Anda sendiri dengan mengubah gambar dan menghubungkannya ke internet.

1. Mengganti Gambar Wallpaper (Latar Belakang)

1. Arahkan mouse ke bagian kosong di halaman depan (Desktop).
2. Klik Kanan, lalu pilih menu paling bawah yaitu Personalize.
3. Di bagian Background, pilih "Picture".
4. Klik tombol Browse untuk mencari foto Anda sendiri, atau klik gambar yang sudah disediakan Windows.
5. Tampilan layar Anda akan langsung berubah!

2. Menghubungkan ke Internet (Wi-Fi)

Di sudut kanan bawah layar (dekat jam dan tanggal), carilah ikon berbentuk sinyal lengkung atau ikon bola dunia (globe).

  • Klik ikon tersebut.
  • Pilih nama Wi-Fi atau Hotspot HP Anda.
  • Klik Connect, lalu masukkan password (kata sandi) Wi-Fi tersebut.
  • Tunggu beberapa saat sampai tulisannya berubah menjadi Connected.

  • Hubungkan komputer ke jaringan internet.
  • Buka browser (Google Chrome), lalu buka situs web google.com. Fotolah layar Anda yang menunjukkan tampilan tersebut.

ðŸ“… Minggu 4: Keamanan & Perawatan Komputer

Minggu terakhir di Modul 1! Sama seperti motor yang butuh ganti oli, komputer juga butuh dibersihkan dari "sampah digital" dan dilindungi dari virus jahat.

1. Melindungi dari Virus (Windows Security)

Windows 10 sudah memiliki Antivirus bawaan gratis yang sangat kuat bernama Windows Defender. Anda tidak perlu mengunduh antivirus lain.

Cara Memindai (Scan) Virus:
1. Klik logo Start (kiri bawah) lalu ketik: Windows Security.
2. Buka aplikasinya, lalu klik Virus & threat protection.
3. Klik tombol biru Quick Scan.
4. Biarkan komputer bekerja memeriksa seluruh file Anda dari ancaman virus.

2. Membersihkan Sampah (Disk Cleanup)

Semakin lama komputer dipakai, semakin banyak file sampah (Cache) yang menumpuk dan membuat komputer jadi lelet/lemot.

  • Klik logo Start (kiri bawah), ketik: Disk Cleanup lalu buka.
  • Pilih Drive (C:), lalu klik OK.
  • Centang semua kotak (seperti Temporary files, Thumbnails, Recycle Bin).
  • Klik OK dan pilih Delete Files. Jangan khawatir, file dokumen, foto, atau video Anda TIDAK akan terhapus. Ini hanya menghapus sampah sistem.

ðŸŽ¯ Tugas Praktik Minggu 4 (Final Modul 1):

  • Jalankan proses Quick Scan di komputer Anda.
  • Jalankan proses Disk Cleanup.
  • Ketik sebuah laporan singkat (di Word atau Notepad) dengan kalimat: "Saya (Nama Anda) telah berhasil memindai virus dan menghapus sampah di komputer".
