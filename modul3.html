Selamat datang di level Intermediate! Penguasaan Excel tingkat lanjut adalah kunci emas untuk lolos tahap wawancara kerja administrasi, keuangan, dan data entri.

ðŸ“… Minggu 1: Review Rumus Dasar & Referensi Absolut ($)

Sebelum masuk ke rumus yang rumit, kita harus memperkuat dasar perhitungan dan memahami cara mengunci sel.

1. Pengingat Rumus Dasar

  • =SUM(A1:A5) : Menjumlahkan angka.
  • =AVERAGE(A1:A5) : Mencari nilai rata-rata.
  • =MAX(A1:A5) & =MIN(A1:A5) : Mencari nilai tertinggi dan terendah.

2. Mengunci Sel dengan Simbol Dollar ($)

Pernahkah Anda menarik (drag) rumus ke bawah, tapi hasilnya malah *error* (Value)? Itu karena referensi selnya ikut turun. Kita harus menguncinya!

Tekan tombol F4 di keyboard setelah mengklik sel pada rumus.
Contoh: =A1*$B$1
Artinya, jika rumus ditarik ke bawah, A1 akan berubah menjadi A2, A3, dst, tetapi B1 akan tetap terkunci di B1.

ðŸŽ¯ Tugas Praktik Minggu 1:

  • Buat tabel "Perhitungan Diskon". Kolom: Nama Barang, Harga Asli, Diskon (%), dan Harga Akhir.
  • Tuliskan angka persentase diskon (misal 10%) di satu sel khusus di luar tabel.
  • Gunakan referensi absolut ($) untuk menghitung Harga Akhir semua barang dengan menarik rumus ke bawah.

ðŸ“… Minggu 2: Fungsi Logika (IF Bertingkat)

Fungsi IF membuat Excel bisa "berpikir" dan mengambil keputusan berdasarkan syarat tertentu.

1. Logika IF Dasar

Struktur: =IF(Syarat, "Jika Benar", "Jika Salah")
Contoh: =IF(C2>=75, "Lulus", "Remedial")
Artinya: Jika nilai di C2 lebih atau sama dengan 75, tulis Lulus. Jika kurang, tulis Remedial.

2. Logika IF Bertingkat (Nested IF)

Bagaimana jika syaratnya lebih dari dua (Misal: A, B, C, D)? Kita masukkan IF di dalam IF.

Contoh Menentukan Grade Nilai:
=IF(C2>=90, "A", IF(C2>=80, "B", IF(C2>=70, "C", "D")))

ðŸŽ¯ Tugas Praktik Minggu 2:

  • Buat tabel "Daftar Nilai Ujian Siswa" (Nama, Nilai Angka, Grade Huruf, Status Kelulusan).
  • Gunakan rumus IF untuk menentukan Status Kelulusan (Lulus jika >=80).
  • Gunakan rumus IF Bertingkat untuk menentukan Grade Huruf (A, B, C, D).

ðŸ“… Minggu 3: Fungsi Pencarian Data (VLOOKUP)

Ini adalah rumus yang paling sering diuji saat melamar kerja. VLOOKUP berfungsi mencari data di tabel lain secara otomatis berdasarkan suatu kata kunci.

Cara Kerja VLOOKUP

Bayangkan Anda adalah kasir. Anda memasukkan Kode Barang (Kata Kunci), dan Excel otomatis memunculkan Nama Barang dari tabel *database* gudang.

Struktur: =VLOOKUP(Sel_Kata_Kunci, Tabel_Database, Kolom_Ke_Berapa, FALSE)

Contoh: =VLOOKUP(A2, $F$2:$H$10, 2, FALSE)
Artinya: Cari kode di A2, temukan di tabel F2 sampai H10 (jangan lupa dikunci $), lalu tampilkan data dari kolom ke-2. (FALSE berarti harus cocok persis).

ðŸŽ¯ Tugas Praktik Minggu 3:

  • Buat dua tabel dalam satu Sheet. Tabel 1 (Kiri) adalah Tabel Transaksi (Kode Barang, Nama Barang, Harga). Tabel 2 (Kanan) adalah Database Barang.
  • Gunakan VLOOKUP untuk mengisi kolom Nama Barang dan Harga di Tabel Transaksi secara otomatis saat Anda mengetik Kode Barang.

ðŸ“… Minggu 4: Manajemen Data & Pivot Table

Ketika Anda memiliki ribuan baris data, rumus saja tidak cukup. Anda butuh Pivot Table untuk merangkum data menjadi laporan dalam hitungan detik.

1. Sort & Filter Data

Blok judul tabel Anda, lalu ke menu Data > Filter. Akan muncul tanda panah di setiap judul untuk menyaring data (misal: hanya tampilkan penjualan bulan Januari).

2. Membuat Pivot Table

1. Klik salah satu sel di dalam tabel besar Anda.
2. Pergi ke menu Insert > PivotTable, lalu klik OK.
3. Di panel sebelah kanan, centang kolom yang ingin dianalisis.
4. Tarik field (misal: "Cabang Toko") ke kotak Rows, dan field "Total Penjualan" ke kotak Values.
5. Excel akan otomatis membuat tabel laporan rekap total penjualan per cabang!

ðŸŽ¯ Tugas Praktik Minggu 4 (Final Modul 3):

  • Buatlah data dummy (palsu) "Laporan Penjualan Toko" sebanyak minimal 30 baris (Kolom: Tanggal, Nama Sales, Cabang Toko, Total Jual).
  • Buat 1 Pivot Table yang menampilkan "Total Penjualan Masing-Masing Sales".
  • Simpan file Excel ini. Pastikan Sheet 1 berisi data mentah, dan Sheet 2 berisi Pivot Table.
