Instruktur: Admin Budi

Selamat datang di level Advance! Di sini Anda akan belajar bahasa "Kerangka" (HTML) dan bahasa "Riasan" (CSS) untuk membuat sebuah halaman website asli.

ðŸ“… Minggu 1: Struktur Dasar HTML

HTML (HyperText Markup Language) bukanlah bahasa pemrograman, melainkan bahasa markup untuk menyusun kerangka konten di web.

1. Persiapan Alat (Text Editor)

Untuk menulis kode, kita butuh aplikasi khusus. Silakan download dan instal Visual Studio Code (VS Code) di komputer Anda (Gratis).

2. Anatomi Tag HTML

Sebagian besar tag HTML memiliki pembuka <tag> dan penutup </tag>.

<!DOCTYPE html>
<html>
  <head>
    <title>Website Pertamaku</title>
  </head>
  <body>
    <h1>Halo Dunia!</h1>
    <p>Ini adalah paragraf pertama saya.</p>
  </body>
</html>

ðŸŽ¯ Tugas Praktik Minggu 1:

  • Buka VS Code, buat file baru bernama index.html.
  • Ketik struktur kode HTML di atas ke dalam file tersebut.
  • Ubah teks "Halo Dunia" menjadi nama Anda. Simpan (Ctrl+S) dan buka file tersebut menggunakan Google Chrome.

ðŸ“… Minggu 2: Gambar, Link, dan Daftar (List)

Website yang isinya hanya teks akan membosankan. Mari kita tambahkan gambar dan link yang bisa diklik.

1. Memasukkan Gambar

Gunakan tag <img>. Tag ini spesial karena tidak butuh penutup.

<img src="foto-saya.jpg" alt="Foto Profil" width="200">

2. Membuat Tautan (Link)

Gunakan tag <a> (Anchor). Atribut href berisi alamat web tujuan.

<a href="https://google.com">Klik untuk buka Google</a>

ðŸŽ¯ Tugas Praktik Minggu 2:

  • Lanjutkan file HTML minggu lalu. Tambahkan sebuah foto (gambar apa saja) di bawah paragraf.
  • Di bawah gambar, buat tulisan "Klik di sini untuk melihat portofolio saya" yang bisa diklik (Link), arahkan link tersebut ke instagram/sosmed Anda.

ðŸ“… Minggu 3: Mempercantik dengan CSS Dasar

HTML adalah kerangkanya, CSS (Cascading Style Sheets) adalah cat dan bajunya. CSS bertugas mengubah warna, ukuran, dan posisi.

Menulis CSS di dalam HTML (Internal CSS)

Kita meletakkan CSS di dalam tag <style> di dalam bagian <head>.

<style>
  body {
    background-color: #f0f8ff;
    font-family: Arial, sans-serif;
  }
  h1 {
    color: blue;
    text-align: center;
  }
</style>

ðŸŽ¯ Tugas Praktik Minggu 3:

  • Tambahkan blok <style> di file HTML Anda.
  • Ubah warna latar belakang web Anda menjadi warna kesukaan Anda (bisa gunakan kata bahasa Inggris seperti: red, yellow, lightgreen).
  • Ubah warna teks Judul (H1) menjadi putih.
